# KIRO Wallet Web Static Build

Render Static Site settings:

- Root Directory: leave empty or use repository root
- Build Command: leave empty
- Publish Directory: `web_static`

Custom domains:

- `kirowallet.com`
- `www.kirowallet.com`

After adding the custom domain in Render, update your domain DNS with the values Render shows, then submit:

- `https://kirowallet.com/sitemap.xml`
